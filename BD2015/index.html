Projeto de Base de Dados

2015/2016 - 1º Semestre

Seja bem-vindo ao seu Bloco de Notas virtual!

Para poder começar a utilizar o seu bloco de notas, faça já login, ou caso ainda não tenha uma conta, registe-se!

Esta aplicação foi desenvolvida no âmbito do curso de Engenharia de Informática e de Computadores, do Instituto Superior Técnico (Campus Taguspark), na unidade curricular de Base de Dados.
O projeto foi realizado pelo grupo 43, do qual fazem parte os alunos João Calado (65909), Félix Barros (73817) e Vanessa Gaspar (73995).
